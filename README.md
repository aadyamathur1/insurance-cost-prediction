# Insurance Cost Prediction

This project predicts medical insurance charges using Machine Learning.

## Features Used

- Age
- Gender
- BMI
- Number of Children
- Smoking Status
- Region
- Obesity Category

## Machine Learning Model

Linear Regression

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Data Scaling
6. Model Training
7. Prediction and Evaluation

## Result

R² Score: 0.804
